serial protocol for arduino mega (dome control)

line terminator: \n
baud rate: 9600
serial communications might fail during rotation of dome - avoid sending commands during this time

-----------------------------------
get status of dome
status
returns ok:
<current_position>
<direction_of_rotation>
<calibration>

current_position		-	current azimuth in degrees
direction_of_rotation	-	0=still, 1=down, -1=up  (right hand)
calibration				-	0=in progress, 1=done, -1=uncalibrated
-----------------------------------
-----------------------------------
dome set azimuth
ds###.#
returns ok:
"dd"		when dome is done with rotating
returns error:
"ec"		error - calibration is not done
-----------------------------------
dome park (go to home position)
dp
returns ok:
"dd"		when dome is done with rotating
returns error:
"ec"		error - calibration is not done
-----------------------------------
dome get current azimuth
dg
returns ok:
###.#	if everything is fine
returns error:
"ec"		error - calibration is not done
-----------------------------------
-----------------------------------
door open
do
returns ok:
"do"		when door is opened
-----------------------------------
door close
dc
returns ok:
"dc"		when door is closed
-----------------------------------
-----------------------------------
set home at current location
hs
returns ok
-----------------------------------
get home position
hg
returns ok:
###.#
-----------------------------------
-----------------------------------
calibration start
cs

"cd"		when calibration is done
-----------------------------------
-----------------------------------
